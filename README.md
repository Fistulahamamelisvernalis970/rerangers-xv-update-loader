## 🕹️ Using the features

The loader offers two main functions to assist your gameplay.

### Automatic selling
The loader identifies low-value items in your inventory. When you enable the auto-sell feature, the script interacts with the game shop to clear your inventory. This keeps your space clear for high-value loot. You do not need to click individual items anymore.

### Aimbot assistance
The aimbot function helps align your view with targets. It operates by reading coordinate data from the game window. You can toggle this feature using a hotkey provided in the settings menu. This adjustment ensures your aim remains steady during combat.

## ❓ Troubleshooting

If you encounter errors, check these common fixes.

- If the loader fails to open, ensure you have the latest version of the .NET Framework installed from Microsoft.
- If the game does not show the tool is working, restart both the game and the loader.
- Disable your antivirus software temporarily if it blocks the loader from launching. Many antivirus programs flag scripts that interact with other games. Ensure you add an exception for this folder to your security list.
- Run the program as an administrator if you have trouble with the auto-sell feature. Right-click the file and select Run as Administrator.
